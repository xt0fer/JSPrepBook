
==== Loops

==== While Loop
[source, js]
----
let x = 1;
while (x &lt 6) {
console.log(x)&semi;
x++;
}
// or
while (playerOne.Alive() == true) {
playerOne.takeTurn();
}
----

Loop through a block of code WHILE condition is true
